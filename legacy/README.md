# Legacy prototypes

Эта папка хранит ранние прототипы. **Они не подключены к сборке и не являются источником правды.**

- `blockduel-demo.html` — самодостаточная HTML-демо с inline-копией игровой логики.
- `blockduel-jsx-prototype/` — React/JSX-прототип компонентов и тем.

Источник правды по правилам — `TZ_BlockDuel_9x9.md`.
Источник правды по коду ядра — `src/core/index.ts`.

Использовать только как референс по визуальным решениям. Не править логику здесь.
