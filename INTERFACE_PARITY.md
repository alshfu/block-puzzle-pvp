# INTERFACE_PARITY — паритет UI Flutter ↔ TS (web-first)

> Сравнение интерфейса Flutter-порта с эталонной TS/React-версией (она всех
> устраивала, «всё работало»). Источник правды по UI — TS-экраны `src/ui/`.
> Цель — довести Flutter-web до уровня TS. Отмечаем `[x]` закрытое, `[~]` в
> работе, `[ ]` пробел. Обновлено: 2026-06-08.

## ✅ Уже на паритете
Меню (hero/режимы/resume/темы/FloatingTheme), игровой экран (доска/рука/таймер/
скорборд/контролы), пауза, конфетти, combo-flash, тосты, **онлайн целиком**
(лобби, матч, reconnect, ремач, ELO-лидерборд), auth (Google web) + Firestore
sync, 3 темы, звук+музыка, маскоты/фон.
